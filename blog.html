Discuss the scope of var, let, and const

var has function or global scope and is hoisted.

let and const have block scope and are not hoisted.

let allows reassignment, while const does not allow reassignment after initial assignment.

Tell us the use cases of null and undefined

null use cases:
Explicitly setting a variable or property to indicate that it has no value or reference
Can be used to reset a variable or property that previously held a value.
Often used when checking for the absence of a value.

undefined Use cases:
Implicitly assigned to variables or properties that have been declared but not initialized.
Represents the default value of function parameters that are not provided with arguments.
Can indicate that a variable or property doesn't hold a valid value.
